# AI 消消乐（100 步道具版）

10×10 网格消消乐 Web 游戏。点击 **「AI 出手」**，算法自动在全部相邻交换中寻找当前最优动作。  
本版本核心特性：**固定 100 步、允许无合并交换、随机道具、任务制通关**。

## 运行方式

需通过本地 HTTP 服务打开（ES Module 不支持 `file://` 直接加载）：

```bash
# 方式一：Python
cd d:\otherwise\AI_best
python -m http.server 8080

# 方式二：npx
npx --yes serve -p 8080
```

浏览器访问：<http://localhost:8080>

## 当前规则

- 棋盘：`10 x 10`
- 图形：圆形、方块、三角形、五角星
- 等级：1 / 2 / 3
- 最大步数：`100`
- 每一步都会消耗步数（即使没有触发合并）

### 合并规则

- 普通格子交换后，若出现横/竖连续 `>=3` 的同形状同等级：
  - 1 级合并：得分 = 参与格子数 × 1，落点升为 2 级
  - 2 级合并：得分 = 参与格子数 × 2，落点升为 3 级
  - 3 级合并：得分 = 参与格子数 × 3，落点清空，并记为该图形一次“三级合并”
- 合并后会下落补格；若再次触发合并，计入连锁分（也计入总分）

### 任务规则（新版）

每局随机指定 2 种目标图形，目标图形任务分各达到 `4` 才通关。  
任务分来源：

1. **三级合并**：同一图形每触发 `1` 次三级合并，任务分 `+1`
2. **道具批量消除**：一次道具消除若总消除格数 `>=9`，对应图形任务分 `+1`

> 注意：道具消除小于 9 格时只加分，不加任务分。

## 道具系统（随机出现）

补格时会随机产生少量道具（约 6% 概率）。

- `column`（列消）：交换后清除道具所在列全部格子
- `bomb`（九宫格）：交换后以目标格为中心清除 `3x3` 区域
- `color`（同形）：交换后清除全局与交换目标同形状的格子

道具消除得分 = 被清除格子总数。

## AI 决策思路（新版）

不是强化学习，使用 **启发式评分 + 两步前瞻**：

1. 枚举所有相邻交换（最多 180 个）
2. 快速排序筛选前 60 个候选（优先目标图形、道具动作）
3. 对候选做完整模拟打分（即时分 / 连锁 / 任务进展 / 道具收益）
4. 再向前看一步（后手候选前 40）作为折扣加分
5. 选总评分最高的交换

评分偏好会随步数动态变化：  
前期更重“布局和后手”，后期更重“立即得分与任务完成”。
