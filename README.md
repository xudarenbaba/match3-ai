# AI 消消乐

10×10 网格消消乐 Web 游戏。点击 **「AI 出手」** 由算法自动选择最优相邻交换。

## 运行方式

需通过本地 HTTP 服务打开（ES Module 不支持 `file://` 直接加载）：

```bash
# 方式一：Python
cd d:\otherwise\AI_best
python -m http.server 8080

# 方式二：npx
npx --yes serve -p 8080
```

浏览器访问：<http://localhost:8080>

## 文件说明

| 文件 | 说明 |
|------|------|
| `game.js` | 规则引擎：合并、下落、连锁、胜负、预演快照 |
| `ai.js` | 启发式 + 2 步向前看的选步算法 |
| `render.js` | 界面渲染、分阶段慢速动画 |
| `index.html` / `style.css` | 页面与样式 |

## 通关条件

每局随机指定 **2 种目标图形**，二者的 **特殊积分**（仅由该图形的 3 级合并产生）均达到 **4** 即通关。

---

## AI 决策思路

不使用强化学习，采用**启发式评分 + 2 步向前看**，所有计算在浏览器内实时完成（< 100ms）。

### 总评分公式

```
总分 = 即时合并分
     + 目标特殊分奖励
     + 连锁分奖励
     + 最优后手分 × 0.55（折扣）
     + 局面评估分
```

### 各模块说明

#### 1. 即时合并分
当前这步交换触发合并所得的分数，作为基础收益。

#### 2. 目标特殊分奖励（权重最高）
- 触发目标图形 3 级合并 → **+5000**
- 触发后恰好通关 → 额外 **+20000**

每局有 2 种目标图形需要各自积累 4 个特殊分，AI 始终将此作为最高优先级。

#### 3. 连锁分奖励
合并引发下落 → 下落后再次触发合并 = 连锁。连锁分额外加权，鼓励 AI 优先选择能引发连锁反应的步骤。

#### 4. 后手潜力（2 步向前看）
执行完本步后，对后续棋盘枚举所有合法交换：
- 先按"是否涉及目标图形"快速排序，目标图形相关的排在前面
- 取排名前 60 个模拟执行，找到最优后手得分
- 以 **× 0.55** 折扣加入本步总分

这保证了 AI 不只看当前一步，还会为下一步制造机会。

#### 5. 局面评估（`evaluateBoard`）
对执行后的棋盘打静态结构分，包含三个指标：

| 指标 | 说明 |
|------|------|
| 各等级格子数量 | 目标图形格子越多、等级越高，分越高 |
| 聚集度（`clusterBonus`） | 统计直接相邻的同等级同图形格子对数量 |
| 近距潜力（`nearMissPotential`） | 同行/列中，同等级同图形格子之间距离 ≤ 2 的对子数量，即"再一步就能凑成 3 连"的机会数 |

**`nearMissPotential` 是关键**：当两个位置都可以合并时，AI 会选合并后周围同类格子更密集、更容易在下一步继续合并的那个，而不是盲目选分数高的。

### 无解时的处理

若整盘所有交换都无法触发合并，自动**洗牌**（保留分数），重新填充棋盘并保证至少存在一个合法交换。
